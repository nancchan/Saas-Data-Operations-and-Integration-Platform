# Salesforce Data Loader

Data Loader is used to import and manage large volumes of customer data in Salesforce.

---

## Use Cases

- Bulk customer import from SaaS system
- Data corrections
- Data migration
- Exporting CRM data

---

## Validation Rules

- Required fields must be present
- Email format must be valid
- Duplicate records must be prevented
- Correct object mapping must be used

---

## Common Errors

- Missing required field
- Duplicate record detected
- Invalid email format
- Incorrect object mapping

---

## Purpose

Ensures accurate and scalable data synchronization between SaaS platform and Salesforce CRM.
